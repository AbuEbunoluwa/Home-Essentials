# 🏠 Home Essentials Marketplace

A full-stack e-commerce application deployed on AWS with a professional DevOps pipeline.

## 🛠️ Tech Stack
- **Backend:** FastAPI (Python)
- **Frontend:** HTML/CSS/JavaScript
- **Database:** PostgreSQL on Kubernetes
- **Containerization:** Docker
- **Registry:** AWS ECR
- **Orchestration:** AWS EKS (Kubernetes)
- **Load Balancer:** AWS ELB
- **Package Manager:** Helm
- **CI/CD:** GitHub Actions
- **Code Quality:** SonarCloud
- **Monitoring:** Prometheus + Grafana

## 📦 Features
- 30 home essential products across multiple categories
- Product filtering by category
- Add to cart and checkout functionality
- Order confirmation system
- RESTful API with FastAPI

## 🚀 Deployment
Built and deployed using a complete DevOps pipeline including automated testing, code quality checks, Docker containerization, and Kubernetes orchestration
